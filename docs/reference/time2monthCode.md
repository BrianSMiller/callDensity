# Lookup month codes from a POSIXct

Lookup the month codes for a given POSIXct, x. MonthCodes are a factor
from '01'-'12' respectively for months from Jan-Dec

## Usage

``` r
time2monthCode(x)
```

## Arguments

- x:

  POSIXct

## Value

Factor containing month codes '01'-'12' for months from Jan-Dec.
