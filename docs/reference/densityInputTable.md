# Show table of inputs into call density estimate

Show table of inputs into call density estimate

## Usage

``` r
densityInputTable(d, separateCommon = TRUE)
```

## Arguments

- d:

  Data.frame containing call density results

- separateCommon:

  Boolean - if true input parameters k, w, SLmean, SLsd will be put into
  a separate table with only a single row.
